# Command Reference

Each Clavix CLI command is documented in its own file under this directory. Every entry covers description, syntax, arguments, flags, inputs, outputs, real-world examples, and notable messages you may encounter.

## Available Commands

### Prompt Optimization
- [clavix improve](improve.md) - Unified prompt optimization (standard or comprehensive depth)

### PRD & Planning
- [clavix prd](prd.md) - Generate Product Requirements Document
- [clavix plan](plan.md) - Create task breakdown from PRD
- [clavix implement](implement.md) - Execute tasks with progress tracking
- [clavix task-complete](task-complete.md) - Mark a task as completed

### Session Management
- [clavix start](start.md) - Enter conversational mode
- [clavix summarize](summarize.md) - Extract requirements from conversation
- [clavix execute](execute.md) - Run saved prompts

### Verification & Completion
- [clavix verify](verify.md) - Post-implementation verification
- [clavix archive](archive.md) - Archive completed projects

### Utility Commands
- [clavix init](init.md) - Initialize Clavix in a project
- [clavix update](update.md) - Update Clavix configuration
- [clavix list](list.md) - List prompts or projects
- [clavix show](show.md) - Display prompt details
- [clavix config](config.md) - Manage configuration
- [clavix version](version.md) - Show version information

### Agent-Only Commands
- [clavix analyze](analyze.md) - Internal prompt analysis (invoked by improve workflow)

Use these documents alongside the guide section for deeper context on configuration and end-to-end workflows.
